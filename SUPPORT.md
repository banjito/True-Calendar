## Getting Started

### How do I create my first event?
1. Open True Calendar
2. Tap the "+" button in the top-right corner
3. Enter your event title and select a date
4. Set the time if needed
5. Add any notes or description
6. Tap "Save" to create your event

### How do I navigate between calendar views?
- **Swipe left/right** on the calendar to move between months
- **Pinch to zoom** to switch between month, week, and day views
- **Tap the "Today" button** to return to the current date

## Frequently Asked Questions

### How do I edit an existing event?
1. Tap on the event you want to edit
2. Tap the "Edit" button
3. Make your changes
4. Tap "Save" to update the event

### How do I delete an event?
1. Tap on the event you want to delete
2. Tap the "Delete" button
3. Confirm the deletion

### How do I set up recurring events?
1. Create a new event or edit an existing one
2. Tap on "Repeat" or "Recurrence"
3. Choose your pattern (daily, weekly, monthly, yearly)
4. Set an end date if desired
5. Save your changes

### Can I sync with other calendars?
Currently, True Calendar works as a standalone app. Cloud sync and integration with other calendar services are planned for future updates.

### How do I backup my data?
Your data is stored locally on your device. To back up:
- Use iCloud backup to include app data
- Export important events manually (feature coming soon)

## Troubleshooting

### App Crashes or Freezes
- **Restart the app**: Close and reopen True Calendar
- **Restart your device**: Turn your iPhone/iPad off and on again
- **Update the app**: Check for updates in the App Store
- **Free up storage**: Ensure you have enough device storage

### Notifications Not Working
1. Check iOS Settings → Notifications → True Calendar
2. Ensure "Allow Notifications" is enabled
3. Check that notification sounds are on
4. Verify your device is not in Do Not Disturb mode
5. Restart your device if issues persist

### Events Not Appearing
- **Check the date**: Make sure you're looking at the correct date/month
- **Refresh the calendar**: Pull down to refresh
- **Check event creation**: Verify the event was saved successfully
- **Search for events**: Use the search function to find specific events

### Performance Issues
- **Close other apps**: Free up device memory
- **Update iOS**: Ensure you're running the latest iOS version
- **Restart device**: Clear temporary files and memory
